# 布局优化更新说明

## 更新内容

根据要求完成了以下优化：

### 1. 🔄 清单切换位置调整
- **原位置**: 在页面头部统计栏下方
- **新位置**: 移动到搜索框上方，独立的容器区域
- **视觉效果**: 更加突出，用户更容易发现和使用

### 2. 🏷️ 标签名称简化
- **平台清单** → **平台**
- **系统清单** → **系统**
- **历史系统清单** → 保持不变（仅在进入历史系统时显示）

### 3. 🔀 标签顺序调整
- **原顺序**: 平台 → 系统 → 历史系统
- **新顺序**: 系统 → 平台 → 历史系统
- **默认显示**: 系统清单（原来是平台清单）

### 4. 📐 网格布局优化
- **原布局**: 一行显示4个系统 (minmax(320px, 1fr))
- **新布局**: 一行显示6个系统 (minmax(200px, 1fr))
- **卡片调整**: 
  - 内边距从 30px 25px 减少到 20px 15px
  - 圆角从 20px 减少到 16px
  - 图标从 3rem 减少到 2.5rem
  - 标题字体从 1.3rem 减少到 1.1rem

### 5. 🎨 界面细节优化
- **收藏按钮**: 从 35px 减少到 30px
- **历史徽章**: 从 35px 减少到 30px
- **系统分类标签**: 内边距和字体大小相应调整
- **系统状态**: 字体大小从 0.9rem 减少到 0.8rem
