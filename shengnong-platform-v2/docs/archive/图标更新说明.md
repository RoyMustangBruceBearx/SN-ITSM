## 🐛 常见问题

### Q1: 图标不显示？
**A:** 检查以下几点：
1. 图标文件是否存在于正确路径
2. 文件名是否正确
3. 浏览器是否缓存了旧图标（清除缓存）
4. 检查浏览器控制台是否有404错误

### Q2: 图标显示模糊？
**A:** 
1. 确保图标文件分辨率足够高
2. 使用 `object-fit: cover` 或 `contain`
3. 提供2x或3x的高清图标

### Q3: 图标变形？
**A:**
1. 确保容器是正方形
2. 使用 `object-fit: cover` 保持比例
3. 检查图标原始比例

### Q4: 如何恢复emoji？
**A:**
1. 将所有 `<img>` 标签替换回 `🐷`
2. 或者保留一份备份代码
