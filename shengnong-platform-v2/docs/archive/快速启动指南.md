# 快速启动指南 - AI配置功能

## 🚀 快速开始

### 1. 启动项目

```bash
cd shengnong-platform-v2
npm install
npm run dev
```

### 2. 访问项目

浏览器打开：`http://localhost:5173`

## 🎯 测试AI配置功能

### 方法一：在首页侧边栏测试

1. **打开AI助手**
   - 点击右下角的AI助手浮动按钮（🐷神农晓问）

2. **开始对话**
   - 在欢迎界面点击"开始对话"按钮

3. **打开配置面板**
   - 在聊天界面头部找到齿轮图标（⚙️）
   - 点击齿轮按钮

4. **配置AI参数**
   - API地址：`http://47.236.87.251:3000/api`
   - 模型选择：`qwen2.5:7b`（或其他模型）
   - 最大令牌数：`4000`
   - 创造性：`0.7`

5. **保存配置**
   - 点击"保存设置"按钮
   - 看到"API设置已保存！"提示

6. **验证持久化**
   - 刷新页面
   - 重新打开配置面板
   - 确认配置已保存

## 🔧 配置说明

### API地址
- **默认值**：`http://47.236.87.251:3000/api`
- **说明**：AI服务的API端点地址
- **格式**：完整的HTTP/HTTPS URL

### 模型选择
- **默认值**：`qwen2.5:7b`
- **可选项**：
  - `qwen2.5:7b` - Qwen2.5 7B模型（推荐）
  - `gpt-3.5-turbo` - GPT-3.5 Turbo
  - `gpt-4` - GPT-4
  - `llama2` - Llama 2

### 最大令牌数
- **默认值**：`4000`
- **范围**：100 - 8000
- **说明**：单次对话的最大令牌数，影响回复长度

### 创造性（Temperature）
- **默认值**：`0.7`
- **范围**：0 - 1
- **说明**：
  - `0.0` - 最保守，回复更确定
  - `0.5` - 平衡
  - `1.0` - 最有创造性，回复更多样

## 🐛 常见问题

### Q1: 配置面板打不开？
**A:** 检查以下几点：
1. 确认已点击"开始对话"进入聊天界面
2. 检查浏览器控制台是否有错误
3. 确认ConfigService已正确初始化

### Q2: 配置保存后不生效？
**A:** 
1. 检查localStorage是否被禁用
2. 查看浏览器控制台的错误信息
3. 尝试清除localStorage后重新配置

### Q3: 刷新页面后配置丢失？
**A:**
1. 检查浏览器是否启用了隐私模式
2. 确认localStorage权限正常
3. 查看控制台是否有存储错误

### Q4: 点击外部无法关闭配置面板？
**A:**
1. 确认点击的是配置面板外部区域
2. 检查事件监听是否正常绑定
3. 尝试按ESC键关闭
